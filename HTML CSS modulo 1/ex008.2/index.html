Código fonte

Uma tag para dar exemplos de código no texto é a code ex: document.getElementById('teste')

o code é melhor utilizado dentro da tag <pre></pre>, caso você queira respeitar a indentação do código.

exemplo de códico em python

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR!')
